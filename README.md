# 🎮 Hangman Game (Python Project)

## 📌 Description
This is a simple text-based Hangman game built using Python. The player guesses letters to find the hidden word.

## ✨ Features
- Multiple difficulty levels (Easy, Medium, Hard)
- Hint system
- Score tracking
- ASCII Hangman drawing
- Replay option

## 🛠 Technologies Used
- Python
- Random module
- Loops & conditions
